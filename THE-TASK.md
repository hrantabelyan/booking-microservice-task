# Тестовое задание: микросервис

# бронирования переговорок

Время на выполнение 2ч.

## Контекст

В компании есть переговорные комнаты. Сотрудники бронируют их для встреч.
Нужно реализовать **микросервис** типа JSON API, который позволяет:
● создавать бронирование переговорки
● получать список бронирований
○ По себе
○ По комнате

## Что сдать

```
● API сервис
● Примеры запросов
● AI можно использовать без ограничений
● Фреймворк/БД любые
● Запушить в github public repo
○ Желательно не 1м коммитом, чтобы были видны изменения после AI
генерации
● Не нужен
○ фронтенд
○ авторизация (просто передаем uid в запросах)
```
